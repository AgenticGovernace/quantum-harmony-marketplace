# Project Scaffolder

Lay down a project's governance frame: a `.agents/` layer with `instructions.md` (immutable scope rules), a per-directory `index.md`, and a seeded `logs/` dir for provenance and reflection. Stackable — run at the repo root or in any subfolder. Run this **before** `agent-definition`.
