# Agent Definition

Build an agent's **Agent Card** (Role, Mission, Output, Escalation, and persistence-gated Memory / Reflection / Audit) and write it to `agents/NAME/AGENTS.md`, inheriting any parent `.agents` governance. Pairs with `project-scaffolder` (frame first) and `agent-scaffolder` (full scaffold).
